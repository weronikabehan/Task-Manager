Detailed Description

Represents a single task with name, status, deadline, etc.

A task holds information such as its name, description, category, completion status, priority, and an optional deadline.

Parameters

Other fields are initialized with default values:

  • done: false
  • priority: prio::Whenever
  • deadline: empty time_point
